Description

给一个主串S以及M个字符串。问这M个字符串在主串中出现多少次。

Input

第一行一个整数N，表示S的长度。

给一个主串S以及M个字符串。问这M个字符串在主串中出现多少次。

下面一行N个字母，即主串S。

Output

一个正整数，表示出现的次数。

3

a

aba
